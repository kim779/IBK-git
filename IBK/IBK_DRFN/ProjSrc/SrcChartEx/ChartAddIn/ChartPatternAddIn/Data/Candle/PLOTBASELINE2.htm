PLOTBASELINE2

Use possible script
- Trade Station
- Yes Language (Hanhwa Securities)


PLOTBASELINE2 It is an output function which expresses the calculated data as index base line.

PLOTBASELINE1~ 3 (
  nExpression,
  strName,
  nColor
);

Parameter 1 nExpression
[nExpression] Enumerate the index and the data and the function which will be displyed on a chart.

Parameter 2 strName
[strName (Optional)] Enumerate the base line name which will be displayed. The defalut is a function name.

Parameter 3 nColor
[nColor(Optional)] Select one among below colors. The default is BLACK.
nColor Description
BLACK 　
BLUE 　
GREEN 　
CYAN 　
RED 　
MAGENTA 　
YELLOW 　
WHITE 　
GRAY

Examples
PLOTBASELINE1(0, "Base line", RED);    // display 0 line
PLOTBASELINE2(80, "Base line1");    // display 80 line
PLOTBASELINE3(20);    //display 80 line,name is "PLOTBASELINE3" as the deflaut

Remarks
- PLOTBASELINE function support 3 functions from PLOTBASELINE1 to PLOTBASELINE3.
- The use of PLOTBASELINE function must use from PLOTBASELINE1 until PLOTBASELINE3 in order...;
- For example,can be used PLOTBASELINE1...;, PLOTBASELINE2...; but if it does not hit to an order,can not use PLOTBASELINE1...;, PLOTBASELINE3...;
