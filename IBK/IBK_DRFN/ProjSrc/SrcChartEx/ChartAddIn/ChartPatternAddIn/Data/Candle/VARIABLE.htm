VARIABLES The reserved word is a declaration to use variable in a function and will be able to store any value in the script.

VARIABLE : nVariable(i);

Variable nVariable
[variable] Enumerate the variable name which it will use from the inside.

Initial Value (i)
Enumerate the initial value of variable.

Examples
VARIABLE : nVariable(CLOSE);
// Declare nVariable from the script inside and the defalut uses close price.
nVariable = OPEN;
// Assigns nVariable to open price.

Remarks
- The inside variable which has become declaration is effective only inside script.
