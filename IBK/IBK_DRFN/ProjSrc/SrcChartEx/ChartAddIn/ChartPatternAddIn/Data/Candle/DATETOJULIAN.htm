DATETOJULIAN The function calculates Julian date of the date when it designates.

DATETOJULIAN (
  nDate
);

Parameter nDate
[nDate] Input the date with the year month and day. 2005 December 1st inputs with 20051201.

Output
[Output] It outputs the Julian date of nDate. The Julian date is based on 1900 and 1900 is 0.

Examples
VARIABLE : nDate(0);
nDate = DATETOJULIAN(20050916);
