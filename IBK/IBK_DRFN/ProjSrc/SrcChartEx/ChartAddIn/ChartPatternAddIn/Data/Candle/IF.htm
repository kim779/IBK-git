IF The reserved word is the conditional statement for a comparison operation.

IF (
  expression
)

Expression expression
Enumerate the comparison arithmetic statement.

Examples
VAR : nVariable(0);
IF (CLOSE > OPEN) THEN
  nVariable = CLOSE;
ELSE
  nVariable = OPEN;
// if close price greater than open price,then assign nVariable to close else assign nVariable to open.
