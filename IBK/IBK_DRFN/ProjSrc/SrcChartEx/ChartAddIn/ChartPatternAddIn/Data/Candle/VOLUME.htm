VOLUME The reserved word uses with the declaration which represent Trading Volume.

VOLUME[n];

Parameters [n]
The default is 0. 0 represents today,1 represents yesterday,2 represents 2 days ago.

Examples
VAR : nVariable(VOLUME);
// Assigns nVariable to today's volumen.
nVarialbe = VOLUME[1];
// Assigns nVariable to yesterday's volume.

Remarks
- Both VOLUME and VOLUME[0] represent today's trading volume.
