Use possible script
- Trade Station
- Yes Language (Hanhwa Securities)


ARRAYThe reserved word is array declaration statement and uses to store several variables.

ARRAY : arValue[n](i);

Variable arValue
[Variable] Enumerate the variable name.

Size [n]
Enumerate the count of the array element which is input.

Initial (i)
It does initially at the price which inputs all elements of the array declared. 선언된 배열의 모든 요소를 입력한 값으로 초기화 합니다.

Examples
ARRAY : arValue[4][(0);
arValue[0] = OPEN;
arValue[1] = HIGH;
arValue[2] = LOW;
arValue[3] = CLOSE;
// to the arValue array,sequencely assign open,high,low,close

Remarks
- The size of array element is limited to 100.
- When array element is [3],the count number of actual use possible array variable is [0], [1], [2] and the first is 0.
