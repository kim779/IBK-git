INPUTS The reserved word is a declaration statement and use to store the passed value when it is called by other scripts.

INPUTS : nInput(i);

Variable nInput
[nInput] Enumerate the variable name which passed by other scripts.

Initial value (i)
When the value passed by other scripts not exist,enumerate the value to use by the default.

Examples
INPUTS : nInput(CLOSE);
// nInput is the value passed from other scripts,the default is close price.

Remarks
- When it is used the value passed by other scripts,a initial value is disregarded.
