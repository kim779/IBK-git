ROUND It is a mathematical function which returns rounding off to the nearest integer of real vaule and position below decimal point as parameters.

Numeric ROUND(
   Numeric nValue,
   Numeric nPrecision
);

Parameter 1 nValue
[nValue] Enumerate the real value which wants knowing round.

Parameter 2 nPrecision
[nPrecision] Enumerate the position below decimal point which does round.

Returns
Returns the nearest integers of the parameters(nValue,nPrecision)

Examples
var: Value1(0);
Value1 = ROUND(10.4357, 3);
// Value1 is 10.346
