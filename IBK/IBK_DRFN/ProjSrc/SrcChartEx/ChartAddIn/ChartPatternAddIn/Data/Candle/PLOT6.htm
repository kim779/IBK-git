- Yes Language (Hanhwa Securities)


PLOT6 It is an output function which expresses the calculated data as index.

PLOT1 ~ 10 (
  nExpression,
  strName,
  nColor
);

Parameter 1 nExpression
[nExpression] Enumerate the index and the data and the function which will be displyed on a chart.

Parameter 2 strName
[strName (Optional)] Enumerate the base line name which will be displayed. The defalut is a function name.

Parameter 3 nColor
[nColor(Optional)] Select one among below colors. The default is BLACK.
nColor Description
BLACK 　
BLUE 　
GREEN 　
CYAN 　
RED 　
MAGENTA 　
YELLOW 　
WHITE 　
GRAY

LGREEN 　
LCYAN

Examples
PLOT1(VOLUME, "VOLUME", RED);    // Display volume as index
PLOT2(MA(CLOSE, 5), "Moving Average");    // Display 5-periods moving average as index
.
.
.
PLOT10(OBV);    //Display OVB as index, Name is "PLOT10" as the default

Remarks
- PLOT function support 10 functions from PLOT1 until PLOT10.
- The use of PLOT function must use from PLOT1 until PLOT10 in order...;
- For example,can be used PLOT1...;, PLOT2...; but if it does not hit to an order,can not use PLOT5...;, PLOT10...;
