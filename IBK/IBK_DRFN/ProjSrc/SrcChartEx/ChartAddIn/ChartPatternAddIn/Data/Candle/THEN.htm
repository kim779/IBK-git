THEN The reserved word is a branch statement in order to be advanced when a conditional statement is true.

IF ( expression ) THEN
  expression2;

EXPRESSION expression2
When the condition is true,enumerate the statement which is advanced.

Examples
VAR : nVariable(0);
IF (CLOSE > OPEN) THEN
  nVariable = CLOSE;
ELSE
  nVariable = OPEN;
// If the close greater than the open,then assigns nVariable to close price,else assigns nVariable to open price.
