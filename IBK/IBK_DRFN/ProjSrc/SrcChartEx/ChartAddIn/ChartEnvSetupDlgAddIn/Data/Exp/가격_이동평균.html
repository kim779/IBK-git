가격 이동평균

개요
가격이동평균은 단순, 가중, 지수, 적응으로 이루어져 있음

작성방법
1) 5일 단순이동평균 = 오늘종가 + 1일전 +2일전 + 3일전 + 4일전종가 / 5
2) 5일 가중이동평균 = 오늘종가*5 + 1일전*4 + 2일전*3 + 3일전*2 + 4일전종가*1 / 5+4+3+2+1
3) 5일 지수이동평균 = 오늘종가*1 + 1일전*(2/3)² + 2일전*(2/3)³ + 3일전*(2/3)⁴ + 4일전종가*(2/3)ⁿ / 1 + (2/3)² + (2/3)³ + (2/3)⁴+ (2/3)ⁿ 'n = 5'

해석 및 활용
총 10개의 기간을 설정 할 수 있으며, 각각에 대한 기간, 기준, 계산을 적용 할 수 있음
