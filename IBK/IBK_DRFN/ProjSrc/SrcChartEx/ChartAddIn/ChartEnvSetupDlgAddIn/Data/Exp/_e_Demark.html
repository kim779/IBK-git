Demark
 

1. Overview

It is an index which sets the target price (support line and resistance line) of tomorrow through today's price.

 

2. Fomula

DeMark Target High = Basic pricde - Low
DeMark Basic Price
IF open > close : basic price = (A + 2 X B+ C) / 2
IF open < close : basic price = (2 X A + B + C) / 2
IF open = close : basic price = (A + B + 2 X C) / 2
DeMark Target Low = Basic - A
(A : High, B : Low, C : Close)
 

3. Use

Use support line and resistance line
When the open starts from forecast target price (between forecast high price and forecast low price),it is interpreted that the forecast high price becomes the resistance line and the forecast low price becomes the support line.

Use trend
When the open price starts from over forecast target price,short-term upward trend signal, When the open price starts from below forecast target price,short-term downward trend signal.
