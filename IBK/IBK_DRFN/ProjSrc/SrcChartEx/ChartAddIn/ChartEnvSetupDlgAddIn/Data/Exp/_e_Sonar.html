SONAR
 

1. Overview

It is a indec to grasp the tendency conversion point of view of stock price. It hints the tendency conversion to precede a stock price.

 

2. Fomula

Sonar = today's moving average - moving average n-days ago
Signal line = n-periods moving average of SONAR
 

3. Use

IF Sonar upward crossed zero line,the point of view for buying. IF Sonar downward crossed zero line,the point of view for selling.
IF Sonar upward crossed Signal line,the point of view for buying. IF Sonar downward crossed Signal line,the point of view for buying.
