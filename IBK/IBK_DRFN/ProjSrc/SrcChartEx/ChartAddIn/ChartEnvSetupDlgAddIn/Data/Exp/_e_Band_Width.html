Band width
 

1. Overview

Band width is calculated and is able to predict the fluctuation of hereafter stock price. If the fluctuation of Band width value became larger, inform imminent the tendency conversion.

Band width is the distance of bands in Bollinger bands.

 

2. Fomula

(Bollingerband Upper band value - Bollingerband lower band value) / Bollingerband middle band value
 

3. Use

It tries to use with the Bollingerbands together,if Band width becames excessive wide(the Band Width value increases) and the stock price is formed from the upper limit band vicinity, sooner or later we can predicts the depreciation of stock price.

if Band width becames narrow(the Band Width value depreciate),it hints sidestep situation. if narrow width comes to be wide again,it will be able to forecast the price's big change sooner or later.
