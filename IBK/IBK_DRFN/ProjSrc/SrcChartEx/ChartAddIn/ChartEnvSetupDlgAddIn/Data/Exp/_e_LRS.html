LRS (Linear Regression Slope)
 

1. Overview

Linear Regression Slope is calculated rectilinear regression line everyday during the past specific periods by the the method which is identical with Linear Regression Indicator and index connected Linear Regression Slope each date.

 

2. Fomula

Y = A + BX
A = 1/N (∑Y - B*∑X)
B = (N*∑XY-∑X*∑Y) / (N*∑X²-(∑X)² )
X : Time flow
N : Periods (Typically 14)
Y : LRL(Liner Regression Indicator)
B : LRS(Liner Regression Slope)
 

3. 적용방법

In case interpreting LRS,it is good to analyze R-squared simultaneously.

When LRS initially has meaning positive value,a point of view for buying. When LRS changes to the depreciation slope,Clearing the buy postion or a point of view for selling
